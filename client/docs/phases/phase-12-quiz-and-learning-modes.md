# Phase 12 - Quiz And Learning Modes

Goal: expand beyond map guessing into geography learning.

## Required Features

- Quiz mode shell.
- Question types:
  - identify country
  - identify capital
  - identify flag
  - identify location from panorama
- Multiple choice answers.
- Timed and untimed quiz settings.
- Score and accuracy summary.
- Review wrong answers.

## Optional Features

- User-created quizzes.
- Live hosted quiz rooms.
- Classroom-friendly room codes.
- Study mode with spaced repetition.

## Done When

- Player can complete a quiz session.
- Results show correct and incorrect answers.
- Quiz data is separate from location map pools.
