# Phase 10 - Ranked And Leaderboards

Goal: add competitive progression after multiplayer is stable.

## Required Features

- Ranked queue.
- Matchmaking by rating.
- Rating changes after ranked matches.
- Divisions or tiers.
- Season model.
- Leaderboards:
  - global
  - friends
  - country/region optional
- Anti-cheat basics.

## Anti-Cheat Basics

- Server chooses locations.
- Server controls timers.
- Server computes scores.
- Hide answer data from client until reveal.
- Track suspicious perfect/fast guesses.

## Done When

- Ranked matches affect rating.
- Leaderboards update after games.
- Basic abuse cases are blocked or logged.
