# Phase 7 - Accounts And Profiles

Goal: persist player progress and prepare for social/multiplayer features.

## Required Features

- Authentication.
- User profile:
  - username
  - avatar
  - country/region optional
- Saved game history.
- Saved best scores.
- Saved favorite maps.
- Privacy settings.

## Backend Scope

- Database for users, games, maps, and scores.
- Server-side validation for submitted game results.
- Rate limiting for public endpoints.

## Done When

- A user can log in, play, leave, return, and see their history.
- Guest mode still works.
- Private user data is not exposed to other users.
