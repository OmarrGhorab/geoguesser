# Phase 9 - Duels

Goal: add competitive head-to-head gameplay.

## Required Features

- 1v1 match.
- Shared rounds.
- Both players guess each round.
- Health or damage system.
- Closer guess deals damage.
- Damage multiplier increases over time.
- Match ends when a player reaches zero health.
- Rematch button.

## Optional Features

- Team duels.
- Spectators.
- Sudden death.

## Done When

- Two players can complete a duel.
- Damage calculations are server-authoritative.
- Draw/timeout behavior is defined.
