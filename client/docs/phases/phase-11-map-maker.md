# Phase 11 - Map Maker

Goal: let users create and publish their own playable maps.

## Required Features

- Create map.
- Add location by clicking map.
- Add location by lat/lng.
- Import JSON/CSV.
- Edit location metadata:
  - name
  - country
  - region
  - heading
  - pitch
  - zoom
  - pano id optional
- Validate panorama coverage.
- Save draft.
- Publish/unpublish.
- Public/private visibility.

## Moderation

- Report map.
- Hide abusive maps.
- Limit uploads/imports per user.

## Done When

- A user can create a map and play it.
- Public maps can be discovered.
- Invalid locations are flagged before publishing.
