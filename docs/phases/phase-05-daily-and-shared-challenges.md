# Phase 5 - Daily And Shared Challenges

Goal: make fixed-seed games that multiple players can compare.

## Required Features

- Deterministic seeded location picker.
- Daily challenge seed based on date.
- Same daily locations for every player.
- Challenge link format.
- Challenge page loads:
  - seed
  - map pool
  - settings
- Prevent changing rules inside a challenge.
- Challenge result summary.

## Optional Features

- Local daily streak.
- Daily countdown.
- Daily leaderboard after accounts exist.

## Done When

- Two browsers opening the same challenge get the same rounds.
- Daily challenge changes once per day.
- Shared challenge links are stable.
