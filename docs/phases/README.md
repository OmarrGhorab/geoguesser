# Geo Game Phase Plan

This folder splits the GeoGuessr-style clone into buildable phases. Each phase should be small enough to implement, test, and ship without dragging in the whole product at once.

## Recommended Order

2. [Phase 2 - Game Settings](./phase-02-game-settings.md)
3. [Phase 3 - Results And History](./phase-03-results-and-history.md)
4. [Phase 4 - Map Pools And Imports](./phase-04-map-pools-and-imports.md)
5. [Phase 5 - Daily And Shared Challenges](./phase-05-daily-and-shared-challenges.md)
6. [Phase 6 - Streak Modes](./phase-06-streak-modes.md)
7. [Phase 7 - Accounts And Profiles](./phase-07-accounts-and-profiles.md)
8. [Phase 8 - Multiplayer Parties](./phase-08-multiplayer-parties.md)
9. [Phase 9 - Duels](./phase-09-duels.md)
10. [Phase 10 - Ranked And Leaderboards](./phase-10-ranked-and-leaderboards.md)
11. [Phase 11 - Map Maker](./phase-11-map-maker.md)
12. [Phase 12 - Quiz And Learning Modes](./phase-12-quiz-and-learning-modes.md)

## Product Rule

Clone the game idea and expected features, not GeoGuessr branding, UI copy, proprietary maps, or paid assets.
